
# Lotto Number Generator

## Overview

This application will generate a set of random lottery numbers.

## Plan

1.  **HTML:** Create the basic structure of the web page, including a title, a container for the numbers, and a button to generate them.
2.  **CSS:** Style the page for a clean and modern look, including a responsive design.
3.  **JavaScript:** Implement the logic to generate the random lottery numbers and display them on the page.

## Features

*   Generate 6 unique random numbers between 1 and 45.
*   Display the generated numbers in a visually appealing way.
*   Responsive design for mobile and desktop.
